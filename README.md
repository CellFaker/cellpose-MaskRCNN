# cellpose-MaskRCNN
MaskRCNN learning code using mask generated by cellpose.  
Generate mask by cellpose from raw data and transrate to numpy dataset for MaskRCNN.   

## Environment
OS : Ubuntu 16.04.7 LTS  
Python : 3.7  
CUDA : 10.1
